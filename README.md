# YumFinder(얌파인더)

## 💡 Introduce
냉장고 속 재료들로 만들 수 있는 레시피를 추천해주는 Application
## ⭐️ Key Function
* 인기 레시피를 북마크 된 순으로 보여줌(최상단 10개 랜덤)
* 냉장고에 있는 재료를 선택하여 검색 -> 재료와 연관된 레시피 제공
* 사용자가 레시피를 직접 만들어 저장 가능
* 레시피 검색 기능 제공
* 카카오 API를 이용한 로그인 기능으로 불필요한 회원가입 절차 생략
## ⚙️ Architecture
* MVVM
## 🏙 Result
랜덤으로 레시피 추천 -> 스와이프를 통해 변경 가능
